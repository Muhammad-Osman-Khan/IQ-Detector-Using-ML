# 🧠 IQ Detector Using ML Algorithms

An interactive cognitive assessment web application that leverages **7 Machine Learning algorithms** to analyze your pattern recognition, shape sequence, and general knowledge skills in real-time.

## What Makes This Unique?

- **7 ML Models Working Together** - Each algorithm analyzes different aspects of your cognitive performance
- **Real-Time Adaptation** - Difficulty adjusts based on your performance
- **Voice Support** - Speak your answers (Chrome/Edge)
- **Instant Feedback** - See which ML model predicts your success

## The 7 ML Algorithms

| # | Algorithm | What It Does |
|---|-----------|--------------|
| 1 | Logistic Regression | Predicts if you'll answer correctly |
| 2 | Ridge Regression | Estimates response time |
| 3 | Random Forest | Recommends which category to focus on |
| 4 | K-Means Clustering | Groups users by ability level |
| 5 | Gradient Boosting | Estimates IQ band |
| 6 | SVM (RBF Kernel) | Classifies learning style |
| 7 | Naive Bayes | Detects cognitive fatigue |
